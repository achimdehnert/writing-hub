# Writing Hub — Dokumentation

**Writing Hub** ist die zentrale Autorenplattform der iil-Plattform.
Sie verbindet KI-gestützte Kreativwerkzeuge mit einem strukturierten Schreib-Workflow.

## Zielgruppen dieser Dokumentation

| Zielgruppe | Einstieg |
|------------|----------|
| **Anwender** (Autoren) | [Schnellstart](user/quickstart.md) |
| **SW-Entwickler** | [Setup & Architektur](dev/setup.md) |
| **IT-Architekten** | [Systemübersicht](arch/overview.md) |

## Kernfunktionen

- **Buchprojekte** — Outline, Kapitel schreiben, Versionen, Export (DOCX/PDF)
- **Ideen-Studio** — KI-Brainstorming, Verfeinern, Premise, Projektanlage
- **Style Lab** — Schreibstil-DNA: DO/DONT/Taboo/Signature Moves
- **WeltenHub-Integration** — Welten, Charaktere, Orte, Szenen via iil-weltenfw
- **Lektorat & Review** — KI-Analyse pro Kapitel, menschliches Feedback
- **Ideen-Import** — Dokument-Upload → KI-Extraktion → Projektanlage
