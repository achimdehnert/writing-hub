# Ideen-Studio (Kreativagent)

Das Ideen-Studio ist ein KI-gestützter Kreativassistent für die Entwicklung von Buchideen.

## Workflow

```
Session starten
    └── Thema / Genre / Tonalität eingeben
         └── KI generiert 3–5 Buchideen (Brainstorming)
              └── Ideen bewerten (★★★★★)
                   └── Idee verfeinern (Detailtiefe erhöhen)
                        └── Premise generieren (1-Satz Kernaussage)
                             └── Projekt anlegen
```

## Session starten

1. Sidebar → **Ideen-Studio**
2. Klicke **Neue Session**
3. Fülle das Formular aus:
   - **Thema/Prompt** — was soll die Geschichte handeln?
   - **Genre** — Fantasy, Thriller, Romance, ...
   - **Tonalität** — dunkel, humorvoll, episch, ...
   - **Style DNA** — optional: welchen Autorenstil anwenden?
4. Klicke **Brainstorming starten**

## Ideen bewerten

Nach dem Brainstorming siehst du 3–5 Ideen mit:
- Titel
- Logline (1-Satz Zusammenfassung)
- Themen, Konflikte, Zielgruppe

Bewerte jede Idee mit 1–5 Sternen.

## Idee verfeinern

Klicke bei einer Idee auf **Verfeinern** — die KI ergänzt:
- Charakterkonzepte
- Weltkonzept
- Akt-Struktur
- Unique Selling Point

## Premise generieren

Eine **Premise** ist der destillierte Kern der Geschichte (1–2 Sätze).
Klicke **Premise generieren** — die KI formuliert sie aus Logline + Verfeinerung.

## Projekt anlegen

Direkt aus der Session heraus: **Als Projekt anlegen** — erstellt ein `BookProject` mit:
- Titel aus der Idee
- Beschreibung = Premise
- Genre übernommen

## Ideen-Import vs. Ideen-Studio

| | Ideen-Import | Ideen-Studio |
|--|--|--|
| **Zweck** | Bestehende Dokumente importieren | Neue Ideen entwickeln |
| **Input** | Datei (DOCX/PDF/MD) | Textprompt |
| **Output** | Strukturierter Import | Buchideen → Projekt |
