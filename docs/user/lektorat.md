# Lektorat & Review

## Review

Das **Review-System** ermöglicht strukturiertes Feedback zu Kapiteln.

### Manuelles Feedback

1. Projekt → **Review** → Kapitel auswählen
2. **+ Feedback hinzufügen**
3. Typ wählen: Positiv / Vorschlag / Problem / Frage
4. Feedback-Text eingeben
5. Optional: Textstelle markieren

### KI-Review

Fünf spezialisierte KI-Agenten stehen zur Verfügung:

| Agent | Fokus |
|-------|-------|
| **Stil-Kritiker** | Schreibstil, Sprache, Ausdrucksweise |
| **Story-Editor** | Handlungslogik, Pacing, Charakterkonsistenz |
| **Lektor** | Fehler, Widersprüche, Schwachstellen |
| **Beta-Leser** | Leserperspektive, emotionale Wirkung |
| **Genre-Experte** | Genre-Konventionen, Erwartungen |

Klicke **KI-Review starten** → Agent wählen → Feedback wird automatisch erstellt.

## Lektorat

Das **Lektorat** analysiert alle Kapitel systematisch auf:

- Konsistenzprobleme
- Logikfehler
- Stilprobleme
- Charakterwidersprüche
- Timeline-Fehler
- Pacing-Probleme

### Lektorat starten

1. Projekt → **Lektorat**
2. **Neue Lektorats-Session starten**
3. Die KI analysiert alle Kapitel sequenziell
4. Ergebnisse nach Schweregrad (Error / Warning / Info) gruppiert
5. Einzelne Probleme als gelöst markieren

## Redaktion (Editing)

Das **Editing**-System erstellt konkrete Verbesserungsvorschläge:
- Original-Textstelle + verbesserter Text
- Erklärung der Änderung
- Vorschlag annehmen oder ablehnen
