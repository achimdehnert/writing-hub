# Schnellstart für Anwender

## 1. Einloggen

Rufe [https://writing.iil.pet](https://writing.iil.pet) auf und melde dich mit deinen Zugangsdaten an.

## 2. Erstes Buchprojekt anlegen

1. Klicke in der Sidebar auf **Projekte**
2. Klicke oben rechts auf **+ Neu**
3. Gib Titel, Genre und eine kurze Beschreibung ein
4. Klicke **Projekt erstellen**

## 3. Outline generieren

In der Projektdetailansicht:

1. Klicke auf **Outline generieren**
2. Die KI erstellt automatisch Kapitel-Strukturen basierend auf Titel und Genre
3. Du kannst Kapitel umbenennen, verschieben und ergänzen

## 4. Kapitel schreiben

1. Klicke auf ein Kapitel in der Outline
2. Wähle **Schreiben** — der Chapter Writer öffnet sich
3. Nutze den **KI-Schreiben**-Button um Inhalte zu generieren
4. Bearbeite und verfeinere den generierten Text

## 5. Ideen mit dem Kreativagenten entwickeln

1. Klicke in der Sidebar auf **Ideen-Studio**
2. Starte eine neue Session
3. Beschreibe dein Thema/Genre/Tonalität
4. Die KI generiert 3–5 Buchideen
5. Verfeinere, bewerte, generiere eine Premise und lege direkt ein Projekt an

## 6. Exportieren

In der Projektdetailansicht → **Export**:

- **DOCX** — Word-Dokument mit Formatierung
- **Markdown** — für weitere Bearbeitung
- **PDF** — via Browser-Druck
