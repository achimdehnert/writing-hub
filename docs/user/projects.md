# Projekte

## Übersicht

Ein **Buchprojekt** ist der zentrale Arbeitsbereich. Es enthält:

- **Outline** — hierarchische Kapitelstruktur
- **Kapitelinhalte** — den eigentlichen Text
- **Versionen/Snapshots** — Sicherungspunkte
- **Review & Lektorat** — Feedback und KI-Analyse
- **Publishing-Profil** — Klappentext, Keywords, Metadaten
- **Export** — DOCX, Markdown

## Projekt anlegen

**Manuell:** Projekte → + Neu → Formular ausfüllen

**Per Import:** Projekte → Importieren → Markdown-Datei hochladen

Das Import-Tool erkennt automatisch:
- Kapitel-Hierarchien (H1/H2 Überschriften)
- Manuskript-Format vs. Planungs-Format
- Kapitelinhalte

## Outline

Die Outline ist eine Baumstruktur aus `OutlineNode`-Elementen.

| Aktion | Beschreibung |
|--------|-------------|
| **Generieren** | KI erstellt Outline aus Titel + Genre |
| **Manuell** | Kapitel selbst anlegen und sortieren |
| **Bearbeiten** | Titel, Zusammenfassung, Notizen pro Kapitel |

## Versionen (Snapshots)

Erstelle jederzeit einen **Snapshot** des aktuellen Stands:

- Projekte → Versionen → + Snapshot erstellen
- Snapshots sind unveränderlich und können verglichen werden
- Nützlich vor großen KI-Generierungen

## Publishing-Profil

Unter **Publishing** kannst du:

- Klappentext schreiben oder KI generieren lassen
- Keywords manuell eingeben oder per KI extrahieren
- Zielgruppe und Vermarktungs-Notizen festhalten
